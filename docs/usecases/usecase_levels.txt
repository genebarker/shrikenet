# Usecase Levels

The usecase levels and icons are adopted from "Writing Effective Use Cases"
by Alistair Cockburn, Object Management Group, Sep 2001.

## High Summary

![High Summary][1]

These hold summary usecases together. Should only be a few of these usecases
for the entire system.

## Summary

![Summary][2]

A summary goal usecase 1) shows context in which the user-goal usecases
operate, and 2) life-cycle sequence of those usecases.
